LECTURA 5 - GITHUB

La plataforma GitHub, es una plataforma de desarrollo colaborativo de software para albergar proyectos utilizando el sistema de control de versiones Git, es decir, GitHub almacena el repositorio de código y brinda herramientas realmente útiles para afianzar un trabajo en equipo dentro de un proyecto y/o contribuir a mejorar el software de los demás.

GitHub, además de ser un servicio de alojamiento de código, ofrece varias herramientas útiles para el trabajo en equipo. Entre las razones más sobresalientes de GitHub podemos encontrar:

  • Versionar tu código.
  • Aprender y experimentar.
  • Contribuir.
  • Trabajo en equipo.
  • Estar informado.
  • Visor de código.
  • Mostrar tus conocimientos.
  • Registro de incidencias.
  • Compatibilidad.
  • Precio.

    • Estas son sólo algunas de las ventajas de trabajar en un repositorio como lo es GitHub.
